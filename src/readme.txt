How to run my app
This app uses the ionic 4 framework.

Ionic:

   Ionic CLI                     : 6.10.1 (/usr/local/lib/node_modules/ionic)
   Ionic Framework               : @ionic/angular 6.10.1


System:

   NodeJS : v12.18.3
   npm    : 6.14.6
   OS     : macOS 
Clone the git repository
git clone https://gitlab.ecs.vuw.ac.nz/kongzhan1/swen325-a1.git
Run the application in your browser
ionic serve listApplication
